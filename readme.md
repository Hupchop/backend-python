# Hupchop Python Backend
Framework : FastApi
Author : Amadi Ifeanyi

## Installation
Simply run ```docker build -t myimage .``` to install.

## Serving Locally
Simply run ```docker run -d --name mycontainer -p 80:80 myimage``` to start development server
